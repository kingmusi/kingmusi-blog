# 事件

## on

为 **dom** 添加事件

参数

| 值类型   | 是否必须 | 说明                                                         |
| -------- | -------- | ------------------------------------------------------------ |
| string   | 必须     | 规定要从被选元素添加的一个或多个事件或命名空间。<br />由空格分隔多个事件值，也可以是数组。必须是有效的事件。 |
| string   | 可选     | 规定只能添加到指定的子元素上的事件处理程序。<br /> `其可以为未来新添加的子元素也加上处理事件` |
| any      | 可选     | 规定传递到函数的额外数据。<br />可从 `event.data` 中取出数据 |
| function | 可选     | 规定当事件发生时运行的函数。                                 |

## one

- 为 `dom` 添加一次性的事件

## off

解除 **dom** 上的事件

参数

| 值类型   | 是否必须 | 说明                                                         |
| -------- | -------- | ------------------------------------------------------------ |
| string   | 必须     | 规定要从被选元素添加的一个或多个事件或命名空间。<br />由空格分隔多个事件值，也可以是数组。必须是有效的事件。 |
| string   | 可选     | 规定只能添加到指定的子元素上的事件处理程序。<br /> `其可以为未来新添加的子元素也加上处理事件` |
| function | 可选     | 规定当事件发生时运行的函数。                                 |
