# 特殊字符编码

HTML 为一些特殊字符，规定了容易记忆的名字，允许通过名字来表示它们，这称为**实体表示法**（entity）

| 字符 | 实体写法   |
| ---- | ---------- |
| `<`  | `&lt;`     |
| `>`  | `&gt;`     |
| `"`  | `&quot;`   |
| `'`  | `&apos;`   |
| 空格 | `&nbsp;`   |
| `&`  | `&amp;`    |
| `©`  | `&copy;`   |
| `#`  | `&num;`    |
| `§`  | `&sect;`   |
| `¥`  | `&yen;`    |
| `$`  | `&dollar;` |
| `£`  | `&pound;`  |
| `¢`  | `&cent;`   |
| `%`  | `&percnt;` |
| `*`  | `$ast;`    |
| `@`  | `&commat;` |
| `^`  | `&Hat;`    |
| `±`  | `&plusmn;` |
