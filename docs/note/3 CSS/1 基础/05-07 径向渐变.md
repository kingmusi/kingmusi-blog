# 径向渐变

## 基本语法

```css
radial-gradient([形状] [尺寸] at [位置], 颜色节点1 [位置], 颜色节点2 [位置], ...)
```

#### 常用参数

| 参数类型 | 示例                                                         | 说明         |
| -------- | ------------------------------------------------------------ | ------------ |
| 形状     | `circle`、`ellipse`                                          | 圆形，椭圆形 |
| 尺寸     | `closest-side`、`farthest-side`、`closest-corner`、`farthest-corner` | 控制渐变范围 |
| 位置     | `at center`、`at top left`、`at 30% 70%`                     | 控制圆心位置 |

> 默认：`ellipse at center`

:::demo

```css
div {
    width: 200px;
    height: 200px;
    background: radial-gradient(circle, yellow, orange, red);
}
```

```html
<div></div>
```

## 尺寸模式

#### 1. closest-side

圆一直扩展到最近的边

:::demo

```css
div {
    width: 200px;
    height: 200px;
    background: radial-gradient(circle closest-side, #fb3, #58a);
}
```

```html
<div></div>
```

:::

#### 2. farthest-corner

扩展到最远的角（最常用）

:::demo

```css
div {
    width: 200px;
    height: 200px;
    background: radial-gradient(circle farthest-corner, #fb3, #58a);
}

```

````html
<div></div>
````

:::

## 重复的径向渐变

```css
repeating-radial-gradient(形状 at 位置, 色1 位置, 色2 位置)
```

:::demo

```css
div {
    width: 200px;
    height: 200px;
    background: repeating-radial-gradient(circle, #fb3 0, #fb3 10px, #58a 10px, #58a 20px);
}
```

```html
<div></div>
```

#### 雷达扫描动画

:::demo

```css
div {
    width: 250px;
    height: 250px;
    border-radius: 50%;
    background:
      radial-gradient(circle,
        rgba(0,255,0,.15) 0%,
        rgba(0,255,0,0) 70%
      ),
      conic-gradient(
        from 0deg,
        rgba(0,255,0,0) 0deg,
        rgba(0,255,0,.8) 20deg,
        rgba(0,255,0,0) 40deg
      );
    animation: radar 3s linear infinite;
}

@keyframes radar {
    0%   { transform: rotate(0); }
    100% { transform: rotate(360deg); }
}

```

```html
<div></div>
```
