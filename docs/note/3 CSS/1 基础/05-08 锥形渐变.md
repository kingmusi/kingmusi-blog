# 锥形渐变

## 基本语法

```css
conic-gradient([从哪个角度开始] at [中心点位置], 颜色节点1 [角度], 颜色节点2 [角度], ...)
```

#### 常用参数

| 参数类型     | 示例                        | 说明                   |
| ------------ | --------------------------- | ---------------------- |
| 起始角度     | `from 0deg` / `from 90deg`  | 渐变从哪个角度开始     |
| 中心点位置   | `at center` / `at top left` | 渐变中心点在哪里       |
| 颜色节点位置 | `yellow 30deg, red 180deg`  | 指定每个颜色切换的角度 |

> 默认：`from 0deg at center`

:::demo

```css
div {
    width: 200px;
    height: 200px;
    background: conic-gradient(yellow, red, blue);
}
```


## 重复的锥形渐变

:::demo

````css
div {
  width: 200px;
  height: 200px;
  background: repeating-conic-gradient(
    #fb3 0deg 20deg,
    #58a 20deg 40deg
  );
}

````

```html
<div></div>
```

## 复杂图案

#### 棋盘

:::demo

```css
div {
    width: 200px;
    height: 200px;
    background: conic-gradient(
        #fff 0deg 90deg,
        #000 90deg 180deg,
        #fff 180deg 270deg,
        #000 270deg 360deg
    );
    background-size: 50px 50px;
}
```

```html
<div></div>
```

:::

#### 分割饼图

:::demo

```css
div {
    width: 200px;
    height: 200px;
    border-radius: 50%;
    background: conic-gradient(
        #ff6b6b 0deg 120deg,
        #4ecdc4 120deg 240deg,
        #1a535c 240deg 360deg
    );
}
```

```html
<div></div>
```
