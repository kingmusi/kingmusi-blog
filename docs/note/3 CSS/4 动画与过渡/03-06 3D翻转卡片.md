# 3D 翻转卡片

## 核心原理

3D 翻转卡片是 CSS 3D 变换中最经典的应用场景。它并不是单一属性的功劳，而是需要构建一个完整的 **3D 空间体系**。

需要三个关键步骤（对应三个层级的元素）：
1.  **舞台（Container）**：设置 `perspective`（视距），开启 3D 视觉透视。
2.  **容器（Wrapper）**：设置 `transform-style: preserve-3d`，确保子元素在 3D 空间中渲染，而不是被压平在 2D 平面上。
3.  **面（Faces）**：设置 `backface-visibility: hidden`，使元素背向用户时隐藏；并且给背面元素设置`transform: rotateX/Y(180deg)`，使背面元素默认旋转 180 度。

## 实现

:::demo
```html
<!-- 舞台 -->
<div class="card-scene">
  <!-- 容器 -->
  <div class="card">
    <!-- 正面元素 -->
    <div class="card-face card-front">Front</div>
    <!-- 背面元素 -->
    <div class="card-face card-back">Back</div>
  </div>
</div>
```

```css
/* 1. 舞台：设置视距 */
.card-scene {
  width: 100px;
  height: 160px;
  perspective: 1000px; /* 视距越小，3D效果越夸张 */
}

/* 2. 翻转体：保持3D空间 */
.card {
  position: relative;
  width: 100%;
  height: 100%;
  transition: transform 0.8s;
  transform-style: preserve-3d; /* 关键：允许子元素在3D空间定位 */
}

/* 触发翻转：这里使用 hover 演示，也可以通过 class 控制 */
.card-scene:hover .card {
  transform: rotateY(180deg);
}

/* 3. 面：公共样式 */
.card-face {
  position: absolute;
  left: 0;
  top: 0;
  width: 100%;
  height: 100%;
  /* 关键：背对屏幕时不可见 */
  backface-visibility: hidden;
  /* 居中演示文字 */
  display: flex;
  justify-content: center;
  align-items: center;
  color: white;
  border-radius: 10px;
}

/* 正面样式 */
.card-front {
  background: linear-gradient(45deg, #ff9a9e, #fad0c4);
}

/* 背面样式 */
.card-back {
  background: linear-gradient(45deg, #a18cd1, #fbc2eb);
  /* 关键：背面默认先旋转180度，这样翻转后它才是正的 */
  transform: rotateY(180deg);
}
```
:::

## 垂直翻转

同样的原理，只需要修改旋转轴为 `X` 轴即可。

```css
.card-scene:hover .card {
  transform: rotateX(180deg);
}

.card-back {
  transform: rotateX(180deg);
}
```
