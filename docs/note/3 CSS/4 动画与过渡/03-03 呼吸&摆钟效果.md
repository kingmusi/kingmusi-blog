# 呼吸&摆钟效果

## animation-direction

|         值          |                   描述                   |
| :-----------------: | :--------------------------------------: |
|      `normal`       |  默认值，动画正常播放（从 0% 到 100%）   |
|      `reverse`      |      动画反向播放（从 100% 到 0%）       |
|     `alternate`     |   动画轮流播放，奇数次正向，偶数次反向   |
| `alternate-reverse` | 动画轮流反向播放，奇数次反向，偶数次正向 |

## 呼吸效果

呼吸效果模拟自然呼吸的节奏，通过缩放变化营造舒缓的视觉体验。

直接写css代码，scale 会平滑的 1.2 过渡到 1，但是随后会**生硬地跳回**原来的 scale，可以通过设置 `alternate` 轮流播放解决

:::demo

```html
<div>kingmusi</div>
```

```css
div {
  width: fit-content;
  height: fir-content;
  transform-origin: center;
  color: pink;
  animation: breathe 2s ease-in-out infinite alternate;
}
@keyframes breathe {
  0% {
    transform: scale(1);
    opacity: 0.7;
  }
  100% {
    transform: scale(1.2);
    opacity: 1;
  }
}
```

:::

## 摆钟效果

:::demo

```html
<div></div>
```

```css
div {
  position: relative;
  width: 4px;
  height: 100px;
  background: pink;
  transform-origin: top center;
  /* 关键代码 */
  animation: swing 2s ease-in-out infinite alternate;

  &::after {
    content: '';
    position: absolute;
    bottom: -20px;
    left: -16px;
    width: 36px;
    height: 36px;
    border-radius: 50%;
    background: pink;
  }
}

@keyframes swing {
  0% {
    transform: rotate(-30deg);
  }
  100% {
    transform: rotate(30deg);
  }
}
```

:::
