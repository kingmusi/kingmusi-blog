# 折角效果

## **45deg** 折角的解决方案

在`切角`的基础上，在实现一个**折角**

```css
div {
  width: 200px;
  aspect-ratio: 5 / 4;
  background: 
    linear-gradient(to left bottom, transparent 50%, rgba(0, 0, 0, .5) 0) no-repeat 100% 0 / 20px 20px,
    linear-gradient(-135deg, transparent 20px, #58a 0);
}
```

:::dom
```html
<div></div>
```

```css
div {
  width: 200px;
  aspect-ratio: 5 / 4;
  background: 
    linear-gradient(to left bottom, transparent 50%, rgba(0, 0, 0, .5) 0) no-repeat 100% 0 / 20px 20px,
    linear-gradient(-135deg, transparent 20px, #58a 0);
}
```
:::

可以看到，两个角并不对齐，虽然它们都是 **20px**，原因是折角不是以对角线为单位的

解决办法：使折角乘以 $\sqrt 2$ ，或者切角除以 $\sqrt 2$ ，具体计算使用 `勾股定理`

:::demo

```css
div {
  width: 200px;
  aspect-ratio: 5 / 4;
  background: 
    linear-gradient(to left bottom, transparent 50%, rgba(0, 0, 0, .5) 0) no-repeat 100% 0 / 20px 20px,
    linear-gradient(-135deg, transparent 14px, #58a 0); /* 改切角是需要改一处，更优 */
}
```

```html
<div></div>
```
:::
