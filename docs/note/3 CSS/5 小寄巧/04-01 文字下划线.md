# 自定义下划线

## 解决方案

使用 **background-image** 和 **渐变** 解决

## 直线

:::demo
```html
<div>
  The only way to <span>get rid of a temptaion</span> is to yield to it
</div>
```

```css
span {
  background: linear-gradient(gray, gray) no-repeat;
  background-size: 100% 1px;
  background-position: 0 1.05em;
}

div {
  padding: 0.5rem;
  font-size: 1.5rem;
  color: #000;
  background: #fff;
}
```
:::

用 **text-shadow** 防止下划线越过文本的降部

:::demo
```html
<div>
  The only way to <span>get rid of a temptaion</span> is to yield to it
</div>
```

```css
span {
  background: linear-gradient(gray, gray) no-repeat;
  background-size: 100% 1px;
  background-position: 0 1.05em;
  /* 追加和背景相同颜色的字体描边，由于字体描边在下划线上图层，所以看着会像撑开了一样 */
  text-shadow: .05em 0 white, -.05em 0 white;
}

div {
  padding: 0.5rem;
  font-size: 1.5rem;
  color: #000;
  background: #fff;
}
```
:::

## 虚线

:::demo
```html
<div>
  The only way to <span>get rid of a temptaion</span> is to yield to it
</div>
```

```css
span {
  background: linear-gradient(90deg, gray 66%, transparent 0) repeat-x;
  background-size: .2em 2px;
  background-position: 0 1.05em;
  /* 追加和背景相同颜色的字体描边，由于字体描边在下划线上图层，所以看着会像撑开了一样 */
  text-shadow: .05em 0 white, -.05em 0 white;
}

div {
  padding: 0.5rem;
  font-size: 1.5rem;
  color: #000;
  background: #fff;
}
```
:::

## 波浪线

重复 `^` 这个形状

:::demo
```html
<div>
  The only way to <span>get rid of a temptaion</span> is to yield to it
</div>
```

```css
span {
  background: linear-gradient(-45deg, transparent 40%, gray 0, gray 60%, transparent 0) 0 1em,
    linear-gradient(45deg, transparent 40%, gray 0, gray 60%, transparent 0) .17em 1em;
  background-repeat: repeat-x;
  background-size: .3em .2em;
  text-shadow: .05em 0 white, -.05em 0 white;
}

div {
  padding: 0.5rem;
  font-size: 1.5rem;
  color: #000;
  background: #fff;
}
```
:::

重复半圆这个形状

:::demo
```html
<div>
  The only way to <span>get rid of a temptaion</span> is to yield to it
</div>
```

```css
span {
  background: 
    linear-gradient(transparent 50%, white 0), /* 遮住圆的一半，需要知道背景色 */
    radial-gradient(transparent 50%, gray 0, gray 74%, transparent 0);
  background-repeat: repeat-x;
  background-position: 0 1em;
  background-size: .3em .3em;
  text-shadow: .05em 0 white, -.05em 0 white;
}

div {
  padding: 0.5rem;
  font-size: 1.5rem;
  color: #000;
  background: #fff;
}
```
:::
