# 字体描边

## `-webkit-text-stroke`

- 单色描边

:::demo
```css
div {
  -webkit-text-stroke: 1px pink;
}
```

```html
<div>欢迎来到木厶笔记存放站</div>
```
:::

- 渐变色描边

:::demo
```css
div {
  position: relative;

  /* 字体颜色，需要是一个渐变色 */
  --text-color: linear-gradient(white, white);
  /* 描边宽度 */
  --text-stroke-width: 2px;
  /* 描边颜色 */
  --text-stroke-color: linear-gradient(red, blue);
}
div::before,
div:after {
  content: attr(data-text);
  display: block;
  background-clip: text;
  color: transparent;
}
div::before {
  position: absolute;
  inset: 0;
  background-image: var(--text-stroke-color);
  -webkit-text-stroke: var(--text-stroke-width);
}
div::after {
  position: relative;
  z-index: 1;
  background-image: var(--text-color);
}
```

```html
<div data-text="欢迎来到木厶笔记存放站"></div>
```
:::
