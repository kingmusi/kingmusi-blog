# 饼图

## 普通饼图

```js
series: [
    {
        type: 'pie',
        data: [
            { value: 10, name: '数据1' },
            { value: 20, name: '数据2' },
            { value: 30, name: '数据3' },
        ]
    }
]
```

## 南丁格尔图

**通过半径区分数据大小**

```js
series: [
    {
        type: 'pie',
        data: [
            { value: 10, name: '数据1' },
            { value: 20, name: '数据2' },
            { value: 30, name: '数据3' },
        ],
        roseType: 'radius'
    }
]
```
