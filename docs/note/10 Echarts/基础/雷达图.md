# 雷达图

## 普通雷达图

```js
// 雷达坐标系
radar:{
    indicator:[
        {name: '坐标1'},
        {name: '坐标2'},
        {name: '坐标3'},
        {name: '坐标4'},
        {name: '坐标5'},
        {name: '坐标6'},
    ]
},
series: [
    {
        type:'radar',
        data:[
            {
                name:'种类1',
                value:[10,20,30,40,50]
            },
            {
                name:'种类2',
                value:[30,10,20,50,40]
            },
        ]

    }
]
```
