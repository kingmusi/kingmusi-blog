# 折线图

## 普通折线图

```js
xAxis: {
    name: '折线类目',
    data: ['类目1', '类目2', '类目3']
},
yAxis: {},
series: [
    {
        name: '种类1',
        type: 'line',
        data: [10, 20, 30]
    }
]
```

## 区域填充

```js
{
    name: '种类1',
    type: 'line',
    data: [10, 20, 30],
    areaStyle:{} // 区域填充
}
```
