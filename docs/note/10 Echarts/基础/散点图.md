# 散点图

## 普通散点图

```js
xAxis: {},
yAxis: {},
series: [
    {
        type:'scatter',
        data:[
            [0,0,20],
            [10,10,40],
            [20,10,50],
            [30,30,30],
        ]
    }
]
```

## 大尺寸散点

```js
series: [
    {
        type:'scatter',
        data:[
            [0,0,20],
            [10,10,40],
            [20,10,50],
            [30,30,30],
        ],
        symbolSize: 30
    }
]
```
